# 우진인테리어 백엔드

LX 하우시스 공식 협력사 우진인테리어(woojin-ch.kr)입니다.
Node.js + Express 기반으로 시공사례 관리, 문의 접수, 어드민 패널 API를 구현했습니다.
프론트엔드는 바닐라 HTML/CSS/JS로 작성했습니다.
백엔드를 처음 배우면서 만든 첫 번째 프로젝트입니다.

---

## 프로젝트 규모

- 개발 기간: 2024.10 ~ 2025.01
- 참여 인원: 1명 (개인 프로젝트)
- 실서비스 운영: woojin-ch.kr

---

## 기술 스택

| 구분            | 사용 기술               | 선택 이유                                                 |
| --------------- | ----------------------- | --------------------------------------------------------- |
| Runtime         | Node.js (ESM) + Express | 백엔드를 처음 배우면서 가장 기본적인 구조로 시작          |
| ORM             | Prisma + PostgreSQL     | 타입 안전한 쿼리와 마이그레이션 관리                      |
| 이미지 처리     | sharp                   | 업로드 이미지를 여러 크기로 리사이즈하기 위해 사용        |
| 이미지 스토리지 | Cloudflare R2 + AWS SDK | S3 호환 API를 그대로 사용하면서 무료 이그레스 비용        |
| 인증            | JWT (Bearer Token)      | API 보호를 위한 기본적인 인증 구조                        |
| 프론트엔드      | Vanilla HTML / CSS / JS | 프레임워크 없이 직접 DOM을 다루며 기초를 익히기 위해 선택 |

---

## 주요 기능

**시공사례**

- 프로젝트 CRUD (제목, 위치, 카테고리, 대표이미지, 상세이미지)
- 카테고리별 필터링 및 목록 조회

**이미지 업로드**

- 단일 / 다중 이미지 업로드
- Sharp로 original · large · medium · thumb 4가지 크기 자동 생성
- Cloudflare R2에 업로드 후 로컬 임시 파일 삭제

**문의**

- 고객 온라인 견적 문의 접수
- 어드민에서 문의 상태 관리 및 메모

**어드민**

- JWT 인증 기반 어드민 API
- 방문자 통계 (일별 방문 수, IP 해시화하여 개인정보 보호)
