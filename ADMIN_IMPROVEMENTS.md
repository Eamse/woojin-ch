# 🎨 어드민 패널 기능 분석 및 개선 제안

**작성일**: 2026-01-20  
**대상**: 우진 창호 관리자 시스템

---

## 📊 현재 구현된 기능

### 1. **프로젝트 관리** (`admin-projects.html`)
- ✅ 프로젝트 CRUD (생성, 조회, 수정, 삭제)
- ✅ 메인 이미지 + 상세 이미지 업로드 (최대 20MB)
- ✅ 견적 내역 관리 (항목별 금액 입력)
- ✅ 프로젝트 정보 (위치, 면적, 연도, 카테고리)

### 2. **갤러리 관리** (`admin-gallery.html`)
- ✅ 프로젝트별 이미지 조회
- ✅ 이미지 URL 복사 (원본/썸네일)
- ✅ 프로젝트 ID 기반 필터링
- ✅ 모달로 이미지 미리보기

### 3. **고객 문의 관리** (`admin-inquiries.html`)
- ✅ 문의 리스트 조회 (신규/진행중/완료)
- ✅ 상태 변경 (new → ing → done → cancel)
- ✅ 관리자 메모 작성
- ✅ 검색 (이름, 연락처)
- ✅ 선택 삭제
- ✅ 통계 (신규/진행중/월별 전체)

### 4. **인증** (`admin-login.html`)
- ✅ JWT 기반 로그인
- ✅ 로그아웃

## 💻 기술적 개선

### 1. **코드 분리**
**현재**: HTML 안에 CSS/JS 모두 포함
**개선**: 
- CSS → 별도 파일 (`admin-common.css`)
- JS → 모듈화 (`admin-utils.js`)

### 2. **공통 컴포넌트**
**중복 코드**: 사이드바가 3개 파일에 반복
**개선**:
```javascript
// admin-sidebar.html 만들어서 include
fetch('/admin/admin-sidebar.html')
  .then(res => res.text())
  .then(html => {
    document.getElementById('sidebar-container').innerHTML = html;
  });
```

### 3. **에러 핸들링 강화**
```javascript
// 현재: try/catch만
// 개선: 중앙 집중식 에러 처리
window.handleError = (error, context) => {
  console.error(`[${context}]`, error);
  showToast(`오류: ${error.message}`, 'error');
  // Sentry 같은 에러 트래킹 도구 연동
};
```
